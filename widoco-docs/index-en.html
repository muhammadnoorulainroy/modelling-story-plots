Actionc back to ToC or Class ToC

IRI: https://example.org/story/action#Action

An event with a deliberate agent. Defined as Action ≡ Event ⊓ ∃evt:hasAgent.⊤, so any event that has an agent gets classified as an action automatically.
is equivalent to
Event c and (has agent op some Thing c)
has sub-classes
Attack action c, Move action c, Speech action c
is in domain of
has patient op, has purpose dp, results in op
